# AI Diet, Fitness and Sleep Planner

## Overview
This project is a command-line based AI system developed using Python that provides personalized diet, fitness, and sleep recommendations. It uses a rule-based decision-making approach to analyze user inputs such as age, weight, height, gender, dietary preference, activity level, and fitness goals.

The system performs health analysis by calculating Body Mass Index (BMI) and estimating body fat percentage. Based on this analysis, it generates a structured diet plan, workout routine, and optimal sleep recommendation. The project integrates multiple aspects of lifestyle management into a single executable program.

---

## Features
- Health analysis using BMI and body fat estimation
- Daily calorie requirement calculation based on user goals
- Protein requirement estimation using standard guidelines
- Personalized diet plans:
  - Vegetarian (tofu, paneer, legumes, greens)
  - Non-vegetarian (eggs, chicken, fish, mutton)
- Portion control based on calorie needs
- Workout recommendations:
  - Goal-based (weight loss, muscle gain, maintenance)
  - Gym and home workout options
- Sleep duration recommendation based on age, activity, and recovery needs
- Fully executable via command line (no GUI required)
